# 🌙 Zad Al-Muslim (زاد المسلم)


**Zad Al-Muslim** is a modern, lightweight, and ad-free Progressive Web App (PWA) designed to be the perfect spiritual companion for Muslims. It features a beautiful glassmorphism UI, smooth animations, and offline capabilities.

## ✨ Features

* **📖 Holy Quran:** Read the Quran with a clean, distraction-free interface.
* **🎧 Quran Audio Player:** Listen to multiple reciters (including Abdul-Basit Abdul-Samad, Mishary Alafasy, etc.) with a smart player that supports background audio, media sessions, and favorites.
* **📿 Smart Masbaha (Rosary):** Digital counter with haptic feedback, customizable daily targets, lifetime statistics, and a unique "Power Saving Mode" for OLED screens.
* **📻 Live Quran Radio:** 24/7 live broadcasts from various Quran radio stations.
* **🤲 Azkar & Hisn Al-Muslim:** Morning, evening, and Ruqyah supplications with interactive counters.
* **🎨 Modern UI:** 3-Mode Theme System (Dark Glass, Light Glass, Plain Dark) that remembers your preference.
* **📱 PWA Ready:** Installable on iOS and Android directly from the browser for a native app-like experience.

## 🛠️ Tech Stack
* **Frontend:** HTML5, Vanilla JavaScript, CSS3 (Custom Glassmorphism Design).
* **Audio Streaming:** HLS.js for live radio and direct MP3 streaming from `mp3quran.net`.
* **Storage:** Native browser `localStorage` for saving settings, bookmarks, and statistics.
* **PWA:** Service Workers and Web App Manifest for offline caching and installation.
