# Quran Tasmee Pro - Product Requirements Document (PRD)

## Project Vision

Build a professional Quran memorization and recitation assistant comparable to Tarteel AI.

The application should provide real-time Quran recitation tracking, mistake detection, memorization analytics, revision scheduling, and intelligent feedback while maintaining a premium Quran reading experience.

---

# Current Status

The application already includes:

* Multiple Mushaf versions
* Quran search
* Audio recitation
* Tafsir support
* Offline PWA support
* Bookmarks
* Voice recognition
* Basic Tasmee mode
* Repeat and playback controls

---

# Primary Goal

Transform the current Tasmee implementation from:


Into:

```text
Quran-Aware Speech Engine
        ↓

        ↓

        ↓
```

# Phase 1 — Core Tasmee Engine

## Feature 1: Word-by-Word Tracking

### Requirements

Track every spoken word individually.

Example:

Expected:

```text
الحمد لله رب العالمين
```

User reads:

```text
الحمد لله رب الكريم
```

Result:

```text
الحمد ✓
لله ✓
رب ✓
العالمين ✗
```

### Deliverables

* Word segmentation engine
* Quran token database
* Real-time word matching
* Accuracy percentage

---

## Feature 2: Missing Word Detection

Detect omitted words.

Example:

Expected:

```text
الحمد لله رب العالمين
```

Spoken:

```text
الحمد رب العالمين
```

Output:

```text
Missing:
لله
```

---

## Feature 3: Extra Word Detection

Detect inserted words.

Example:

```text
الحمد لله الكريم رب العالمين
```

Output:

```text
Extra word:
الكريم
```

---

## Feature 4: Ayah Progress Tracking

Automatically determine:

* Current Surah
* Current Ayah
* Current Word Position

Requirements:

* No manual ayah selection
* Continuous progression
* Auto-advance

---

## Feature 5: Real-Time Highlighting

While reciting:

* Highlight current word
* Highlight current ayah
* Animate progression

Status colors:

```text
Green  = Correct
Yellow = Similar
Red    = Incorrect
Gray   = Not Recited Yet
```
